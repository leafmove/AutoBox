## 🎯 这个项目是做什么的？

**叶动智能**是一个基于网页的智能助手平台，就像你的个人AI工作台。它集成了多个实用工具，可以帮你：

- 💬 **助手管理**：随时向智能助手提问，布置执行任务，远程遥控助手
- 📱 **能力拓展**：像逛应用商店一样，使用各种AI应用工具，支持Skills技能增强
- 🔧 **实时编排**：使用强大编辑平台，实时扩展AI智能体能力，简单拖拽或者编码均可
- 🌐 **远程连接**：方便地连接和管理远程服务器,实时监控助手运行环境

## ✨ 主要功能

### 1. 智能助手 💬
- 和AI进行实时对话，像聊天一样简单
- 支持多种内容展示：文字、图表、数学公式、代码高亮等
- 对话记录自动保存，随时可以查看历史聊天

### 2. 应用中心 📱
- 提供丰富的AI应用，就像手机应用商店一样
- 每个应用都有特定功能，满足不同需求
- 权限管理，确保安全使用

### 3. 远程连接 🌐
- 轻松连接远程服务器
- 安全的远程操作体验
- 支持多种连接方式

### 4. 个性化设置 ⚙️
- 浅色/深色主题切换
- 个人中心管理
- 系统设置自定义
